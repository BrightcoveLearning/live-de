Einführung

Mit dem Live-Planer können Sie Aktionen für planen SEP-Jobs (Static Entry Point).. Sie richten einfach einen Scheduler-Workflow mit den Zeiten ein, und die erforderlichen API-Aktionen werden automatisch ausgeführt. Es gibt zwei Arten von Scheduler-Workflows, die Sie einrichten können, wie in den folgenden Abschnitten beschrieben.

Planer starten / stoppen

Mit dem Scheduler können Sie Aktivierungs- und Deaktivierungszeiten für einen SEP-Job einrichten und die Zeiten aktualisieren. Siehe Details in Planen der Aktivierung / Deaktivierung eines SEP-Streams.

Clip Scheduler

Sie können auch die Generierung eines Clips aus Ihrem Live-Job planen und den Zeitplan später aktualisieren oder den Clip abbrechen, wenn sich etwas ändert. Siehe Details in Planen der Clip-Generierung für einen Live-Stream.
