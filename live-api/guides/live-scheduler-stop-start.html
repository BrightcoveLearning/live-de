Bedarf

  • Der Live-API-Schlüssel für Ihr Konto.
  • In Ihrem Live-Konto muss die Scheduler-Überschreibung aktiviert sein.
  • Ein bestehender Live Job mit "statisch": wahr (sehen Statische Einstiegspunkt-Jobs für mehr Informationen).
Beachten Sie, dass Sie die Aktivierung und Deaktivierung auch im Live-Modul planen können - siehe Automatischer Start/Stopp von Live-Events im Kontrollraum für Details

Erstellen Sie einen Scheduler

Nehmen wir an, wir möchten ein 2-stündiges Ereignis planen, das bei 2021-04-02T 14:00:00 + 00:00 beginnt.

Zeitstempel müssen in UTC als Unix-Zeitstempel in Sekunden formatiert sein. Sie können verwenden https://www.unixtimestamp.com/ zur Umwandlung.

Beachten Sie, dass die beschriebenen Aktivierungszeitbeschränkungen Hier gelten weiterhin für geplante Aktivierungen, daher empfehlen wir, eine Zeit einzuplanen, die mindestens 60 Sekunden vor der gewünschten Startzeit liegt.

In diesem Beispiel wählen wir 5 Minuten vor der beabsichtigten Startzeit:

beginne: 2021-04-02T 13:55:00 + 00:00 = 1617386100 Halt: 2021-04-02T 16:00:00 + 00:00 = 1617393600

Führen Sie die folgenden Schritte aus, um den Scheduler zu implementieren POST Anfrage:

https://api.bcovlive.io/v1/scheduler/jobstartstop

Überschriften

Content-Type: application/json
		X-API-KEY: {your API Key}

Text anfordern

{
			"job_id": "your Job ID",
			"activation": {
				"time_utc": 1617386100
			},
			"deactivation": {
				"time_utc": 1617393600
			},
			"description": "My Wednesday broadcast"
		}

Beispielantwort

 
{
	"account_id": "1752604059001",
	"activation": {
		"state": "pending",
		"time_utc": 1617386100
	},
	"deactivation": {
		"state": "pending",
		"time_utc": 1617393600
	},
	"description": "My Wednesday broadcast",
	"job_id": "15981d93877e490185e6ebc9af2c79a3",
	"type": "jobstartstop",
	"workflow_id": "d44cf29f4f184757ac0995beb8a0097e"
}

Aktualisieren Sie einen Scheduler

Sie können die aktualisieren Aktivierung und/oder Deaktivierung Zeit für einen Scheduler, vorausgesetzt, der Job ist in a steht aus Zustand ("zustand": "ausstehend"). Senden Sie dazu eine STELLEN Anfrage zu:

https://api.bcovlive.io/v1/scheduler/jobstartstop/{workflow_id}

Die Überschriften sind die gleichen wie für die POST oben gezeigte Anfrage.

Text anfordern

Fügen Sie im Anfragetextobjekt nur die Aktivierung und/oder Deaktivierung Felder.

So beenden Sie das Ereignis beispielsweise 30 Minuten früher als geplant:

Text anfordern

{
	"deactivation": {
		"time_utc": 1617391800
	}
}

Beispielantwort

 
{
	"account_id": "1752604059001",
	"activation": {
		"state": "pending",
		"time_utc": 1617386100
	},
	"deactivation": {
		"state": "pending",
		"time_utc": 1617391800
	},
	"description": "My Wednesday broadcast",
	"job_id": "15981d93877e490185e6ebc9af2c79a3",
	"type": "jobstartstop",
	"workflow_id": "d44cf29f4f184757ac0995beb8a0097e"
}

Brechen Sie einen Scheduler ab

Sie können die Aktivierung und Deaktivierung mit a abbrechen LÖSCHEN Anfrage:

https://api.bcovlive.io/v1/scheduler/jobstartstop/{workflow_id}

Sie können dies auch nach der Aktivierung tun, um die Deaktivierung einfach abzubrechen.

Überschriften

Der einzige erforderliche Header ist:

X-API-KEY: your API Key

Text anfordern

Für diese Anfrage gibt es keinen Anfragetext.

Beispielantwort

 
{
		"account_id": "1752604059001",
		"activation": {
			"state": "cancelled",
			"time_utc": 1617386100
		},
		"deactivation": {
			"state": "cancelled",
			"time_utc": 1617391800
		},
		"description": "My Wednesday broadcast",
		"job_id": "15981d93877e490185e6ebc9af2c79a3",
		"type": "jobstartstop",
		"workflow_id": "d44cf29f4f184757ac0995beb8a0097e"
	}

Benachrichtigungen

Optional können Sie Benachrichtigungen beim Erstellen des Workflows konfigurieren. Sie müssen eine URL für unseren Service angeben POST zu. Wir senden eine Benachrichtigung, wenn der Scheduler die Aktivierungs- und Deaktivierungsanrufe tätigt. Sie können auch so konfigurieren, dass Sie eine Benachrichtigung erhalten n Sekunden vor der Aktivierung und/oder Deaktivierung. Siehe die Live Scheduler-Benachrichtigungen für mehr Details.

Beispielanforderungshauptteil

 
{
		"job_id": "your Job ID",
		"activation": {
			"time_utc": 1612274100,
			"notification": 600
		},
		"deactivation": {
			"time_utc": 1612281600,
			"notification": 600
		},
		"description": "My Tuesday broadcast",
		"notification_url": "http://example.com/live/jobstartstop/callbacks"
	}
