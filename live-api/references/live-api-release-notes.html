Live-API-Versionshinweise

Dies ist eine Zusammenfassung der Änderungen an der Live-API.

2019-05-13

  • Hinzugefügt color_full_width zu outputs Felder für einen Live-Job schaffen.

2019-05-06

  • Unterstützung für hinzugefügt Hinzufügen von Anzeigenmetadaten zu einem laufenden Live-Stream.

2019-01-28

  • Unterstützung für hinzugefügt RTMP-Ausgänge.

2019-01-21

  • Unterstützung für hinzugefügt neue Eingabeprotokolle (um MPEG2-TS-Eingabe zu ermöglichen).
