Einführung

Wenn Sie Live-Ereignisse mit dem Live-Modul erstellen, müssen Sie ein Live-Aufnahmeprofil auswählen, das die Anzahl und Qualität der von Brightcove Live erstellten Wiedergaben bestimmt. Dieses Thema enthält detaillierte Informationen zu jedem Standard Live Live-Aufnahmeprofil.

Anmerkungen:
  • Für Jobs, die über die Live-API erstellt wurden, können Sie derzeit kein Aufnahmeprofil angeben, sondern müssen stattdessen die Ausgabeeigenschaften definieren.
  • Diese Profile werden speziell beim Erstellen von Live-Events verwendet. Sie werden beim Hochladen von Videos in die Video Cloud nicht verwendet.

Es ist auch möglich, eigene benutzerdefinierte Live-Aufnahmeprofile zu erstellen. Weitere Informationen finden Sie unter Erstellen von benutzerdefinierten Live-Aufnahmeprofilen.
