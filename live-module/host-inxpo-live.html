Einführung

INXPO ist eine Video-Streaming-Plattform zum Erstellen von Event-Erlebnissen im TV-Stil. Um ein größeres Publikum zu erreichen, können Sie Brightcove Live nutzen, um Ihre Veranstaltung zu streamen.

Schritte

Führen Sie die folgenden Schritte aus, um ein Meeting für Brightcove Live auszurichten:

  1. Erstellen Sie in Video Cloud Studio eine Live-Event mit dem Live-Modul.
    Wenn Sie die Übertragung nach Abschluss speichern möchten, überprüfen Sie die Event in Video-Asset umwandeln, wenn abgeschlossen Checkbox!

  2. Kopieren Sie nach dem Erstellen des Live-Ereignisses die Streaming-RTMP-URL und den Streaming-Schlüssel aus dem Kontrollraum Panel.

    • Streaming RTMP URL - Streaming-Endpunkt-URL (Die Endpunkt-URL finden Sie im Brightcove Live Control Room
    • Streaming Key - Stream Name (In Brightcove lautet der Stream-Name immer "am Leben")
  3. Veröffentlichen und binden Sie den Player für dieses Live-Ereignis in Ihre Website / Intranetseite ein und notieren Sie sich die Seiten-URL.

Konfigurieren Sie die Streaming-Sitzung

Mit der Streaming-RTMP-URL und dem Streaming-Schlüssel müssen Sie diese im INXPO-Producer-Tool konfigurieren.

  1. Gehen Sie zum INXPO-Producer-Tool und klicken Sie auf das 3-Punkte-Symbol, um zum Abschnitt Ausgabedetails zu gelangen.
  2. Klicken Sie auf das Stiftsymbol.
  3. Füllen Sie die folgenden Details mit Ihren Brightcove Live-Ereignisanmeldeinformationen aus.
  4. In dem Titel Fügen Sie im Feld den Brightcove Live Stream-Namen hinzu.
  5. In dem Kodierungsprofil Wählen Sie im Feld das Standardcodierungsprofil aus, dessen Einstellungen definieren, wie der Encoder den Webcast komprimiert und an die Teilnehmer sendet.
  6. In dem RTMP-URL Fügen Sie im Feld die URL des Brightcove-Streaming-Endpunkts ein.
  7. Drücke den OK Knopf, und das war's. Du lebst!
