Integrieren eines Zoom-Meetings in Brightcove Live

Mit der Brightcove Live Zoom-Integration können Sie Zoom-Besprechungen einfach mit Brightcove Live übertragen.

Voraussetzungen

Auf dem Zoom-Marktplatz

Die Zoom-Integration mit Brightcove Live wird von der verwaltet Brightcove Live Connect App im Zoom-Marktplatz. Es handelt sich um eine App auf Kontoebene, die von lizenzierten Zoom-Konten auf Eigentümer- oder Administratorebene verwendet werden kann. Stellen Sie sicher, dass Ihr Zoom-Konto über die richtigen Berechtigungen zur Unterstützung der Integration verfügt.

In der Brightcove Video Cloud

Bevor Sie eine Integration starten, müssen Sie über ein benutzerdefiniertes Metadatenfeld verfügen, um zu verfolgen, ob ein Ereignis von Zoom gestreamt wird. Dadurch wird der Status der Integration zwischen Sitzungen verfolgt, dh wenn ein Ereignis derzeit mit einer Zoom-Besprechung verknüpft ist. Weitere Informationen zum Hinzufügen des benutzerdefinierten Metadatenfelds finden Sie unter folgende Dokumentation.

  1. Gehe zu Video Cloud Studio > Administrator > Videofelder > Benutzerdefiniertes Feld hinzufügen
    2. .
  2. Schreib ein Anzeigename für Ihr benutzerdefiniertes Metadatenfeld.
  3. Für die Interner Name Feld sollten Sie schreiben: bc_meeting_id
    Fügen Sie dem benutzerdefinierten Feld keine externen Zoom-IDs oder Daten hinzu.
  4. Schreib ein Beschreibung für Ihr benutzerdefiniertes Metadatenfeld.

Empfohlene Vorgehensweise

Für beste Ergebnisse folgen Sie bitte den Anweisungen unten.

  1. Verwenden Sie die Start und Halt Broadcasting-Schaltflächen im Kontrollraum und nicht im Host-Zoom-Player.
  2. Vermeiden Sie das gleichzeitige Streamen mehrerer Zoom-Besprechungen unter einem einzigen Konto.

Erstellen eines Zoom-Meetings

  1. Greifen Sie zuerst auf Ihre Zoom-Konto und erstellen Sie eine Zoom-Besprechung. Informationen zum Erstellen eines Zoom-Meetings finden Sie unter Zoom-Dokumentation.
  2. Gehen Sie in Zoom zu die Einstellungen > Besprechung (Fortgeschrittene) > Live-Streaming-Meetings.
  3. Stellen Sie sicher, dass Ihre Zoom-Kontoeinstellungen die haben Benutzerdefinierter Live-Streaming-Dienst aktiviert.

Integrieren des Zoom-Kontos in Brightcove Live

Gehen Sie folgendermaßen vor, um ein Zoom-Konto in Brightcove Live zu integrieren:

  1. Melden Sie sich bei Video Cloud Studio an.
  2. Drücke den Administrator Schaltfläche in der Navigationsüberschrift.
  3. Wähle aus Integrationen Option aus dem Dropdown-Menü.
  4. Drücke den Integration hinzufügen Klicken Sie auf die Schaltfläche, um ein oder mehrere Zoom-Konten mit Brightcove Video Cloud zu verknüpfen.
  5. Ein Popup-Fenster mit der Aufforderung, auf Ihr Zoom-Konto zuzugreifen, wird angezeigt. Drücke den Autorisieren Taste.
  6. Geben Sie Ihre Zoom-Anmeldeinformationen ein, um das Konto zu integrieren.

Hinzufügen der Zoom-Integration zu einem Brightcove Live-Ereignis

Gehen Sie folgendermaßen vor, um die Zoom-Integration zum Brightcove Live-Ereignis hinzuzufügen:

  1. Navigieren Sie zu Live-Modul in der Video Cloud.
  2. Drücke den Ereignis erstellen Schaltfläche auf der Live Ereignisse verwalten Seite. Vollständige Informationen zum Erstellen eines Live-Events finden Sie unter Erstellen und Verwalten von Live-Ereignissen mit dem Live-Modul Dokumentation.
  3. Füllen Sie die Details des Live-Ereignisses wie gewohnt aus, bis Sie das erreichen Integrationen Sektion.
  4. Drücke den Integration hinzufügen Taste.
  5. Sie können eine Liste Ihrer aktiven und geplanten Zoom-Besprechungen anzeigen.
  6. Klicken Sie auf das Dropdown-Menü.
  7. Wählen Sie Ihr Meeting aus. In diesem Beispiel wähle ich die Test Docs Meeting.
    Einem Ereignis kann jeweils nur eine Integration hinzugefügt werden. Um die Integration zu löschen, klicken Sie auf Integration entfernen Taste. Wenn Sie mehr als ein Konto haben möchten, können Sie dies im Administrator> Integrationen Tafel.
  8. Wenn Sie bereit sind, das Live-Ereignis zu erstellen, klicken Sie auf Ereignis erstellen Taste.
  9. In dem Kontrollraum Es gibt eine Option mit Registerkarten, mit der Sie Ihre Besprechung über Zoom streamen können. Wenn Sie die Integrationsquelle nicht verwenden möchten, wechseln Sie zurück zur Registerkarte URL aufnehmen oben links.
  10. Wenn Ihr Zoom-Meeting kürzlich erstellt wurde, versuchen Sie erneut, mit Zoom zu synchronisieren Synchronisieren Taste.
    Wenn Ihr Meeting nach der Synchronisierung immer noch nicht angezeigt wird, überprüfen Sie, ob Ihr Meeting Geplant in Zoom und dass Sie weniger als 30 Meetings geplant haben.
  11. Gehen Sie weiter zu Ihrem Zoom-Meeting Konto.
  12. Klicken Sie auf das Blau Start Schaltfläche, um Ihr Meeting zu starten.
    Aufgrund einer Zoom-Beschränkung muss das Zoom-Meeting gestartet werden, bevor Sie mit der Übertragung des Zoom-Meetings an Brightcove Live beginnen.
  13. Zurück im Brightcove Live Kontrollraum , drücke den Übertragung starten Taste.
  14. Nach dem letzten Schritt können Sie Ihr Zoom-Meeting im Control Room sehen.
    • Während des Zoom-Meetings können Sie die Übertragung jederzeit stoppen und neu starten. Wenn Sie das Ereignis jedoch nicht aktiv senden, tritt möglicherweise eine Zeitüberschreitung auf, wenn die Lücke die Zeit für die erneute Verbindung des Ereignisses überschreitet.
    • Die Zoomübertragung über Brightcove wird um ca. 20 Sekunden verzögert.
    • Wenn das Zoom-Meeting nach 20 Sekunden nicht mit dem Streaming beginnt, können Sie auf klicken Stoppen Sie die Übertragung und dann Starten Sie die Übertragung nochmal.
    • Da es sich bei dem Zoom-Stream um einen Dienst eines Drittanbieters handelt, werden einige erweiterte Live-Funktionen möglicherweise nicht unterstützt, einschließlich SSAI.

Deinstallation der Zoom-Integration

Entfernen Sie die Integration aus der Video Cloud

So entfernen Sie Zoom-Integrationen aus Ihrem Konto in Brightcove:

  1. Einloggen in Video Cloud Studio.
  2. Drücke den Administrator Schaltfläche in der Navigationsüberschrift.
  3. Wähle aus Integrationen Option aus dem Dropdown-Menü.
  4. Drücke den Müll Symbol.
  5. Wenn Sie zur Bestätigung aufgefordert werden, klicken Sie auf Bestätigen Taste.

So entfernen Sie Brightcove Live Connect aus Ihrem Zoom-Konto

Sie können die vollständig entfernen Brightcove Live Connect App von Ihrem Zoom-Konto durch Herstellen einer Verbindung zu Zoom:

  1. Melden Sie sich bei Ihrem Zoom-Konto an und navigieren Sie zu Zoom App Marktplatz.
  2. Klicken Verwalten> Installierte Apps oder suchen Sie nach dem Helle Bucht App.
  3. Drücke den Helle Bucht App.
  4. Klicken Deinstallieren.
