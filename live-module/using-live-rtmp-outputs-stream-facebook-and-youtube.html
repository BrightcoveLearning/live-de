Verwenden von Live-RTMP-Ausgaben zum Streamen auf Facebook und YouTube

In diesem Thema erfahren Sie, wie Sie RTMP-Ausgaben im Live-Modul verwenden, um Live-Ereignisse auf Facebook und YouTube zu streamen.

Einführung

Das Live-Modul kann verwendet werden, um ein Live-Ereignis sowohl auf Desktop- als auch auf Mobilgeräten zu übertragen. Nachdem Sie Ihre Ereignisinformationen in das Live-Modul eingegeben haben, erhalten Sie eine Reihe von Encodereinstellungen, die Sie an Ihren Encoder vor Ort anschließen. Normalerweise wird der Live-Stream in einem Brightcove-Player veröffentlicht. Auf der Seite Kontrollraum des Live-Moduls können Sie zusätzliche RTMP-Ausgabe-URLs eingeben. Mit den RTMP-Ausgabeeinstellungen kann der Live-Stream an Social Media-Sites gesendet werden. Beachten Sie, dass die RTMP-Ausgabestunden mit den Ereignisstunden abgerechnet werden. In diesem Thema werden die Schritte zum Erstellen eines einfachen Live-Ereignisses behandelt und anschließend gezeigt, wie die RTMP-Ausgaben zum Streamen eines Live-Ereignisses auf Facebook und YouTube verwendet werden können.

  • Anzeigen, einschließlich serverseitiger Anzeigen (SSAI), werden nicht an RTMP-Ausgaben gestreamt. Stattdessen wird der werbefreie Stream verwendet.
  • RTMP-Ausgänge sind auf 5 Ausgänge pro Live-Job begrenzt. Diese Einschränkung umfasst Verbindungen, die als Fehler markiert sind.

Neues Ereignis erstellen

Führen Sie die folgenden Schritte aus, um ein neues Ereignis zu erstellen.

  1. Öffnen Sie das Live-Modul.
  2. Klicken Sie auf der Seite Ereignisse verwalten auf Ereignis erstellen Schaltfläche oder klicken Live-Event erstellen oder Live-Kanal erstellen in der linken Navigation. Die Seite "Live-Ereignis erstellen" oder "Live-Kanal erstellen" wird geöffnet.
  3. Geben Sie die Veranstaltungsname und alle erforderlichen benutzerdefinierten Felder.
  4. Wählen Sie a Live-Aufnahmeprofil.
  5. Wählen Konvertieren Sie das Ereignis nach Abschluss in ein Video-Asset um eine VOD-Version des Live-Videos in Ihrem Video Cloud-Konto zu speichern.
  6. Klicken ERWEITERTE OPTIONEN und konfigurieren Sie die erweiterten Optionen entsprechend.
  7. Klicken Starten Sie das Streaming um das Live-Event zu starten. Die Kontrollraumseite wird mit Details zum Live-Event einschließlich des geöffnet Streaming-Endpunkt (RTMP-URL) und Stream Name welches von Ihrem Encoder verwendet wird.
    Hinweis: Es dauert ungefähr eine Minute, um Ihre Streams zu verarbeiten.
    Hinweis: Standardmäßig muss der Encoder innerhalb von 30 Minuten nach der Erstellung eines neuen Ereignisses gestartet werden.
  8. Starten Sie das Streaming vom Encoder und der Live-Stream wird auf der Seite Kontrollraum angezeigt.

Sobald das Live-Event gestartet wurde, kann der Live-Stream an gesendet werden Facebook und / oder Youtube.

Hinweis: Stellen Sie sicher, dass der Encoder geschlossen / gestoppt ist, wenn der Live-Stream abgeschlossen ist. Wenn eine Verbindung zwischen einem Encoder und Brightcove geöffnet wird, kann es zu unerwarteten Verbindungsgebühren führen, selbst wenn keine Daten gesendet werden.

Streaming auf Facebook Live

Facebook Live ist ein wohnen Video-Streaming-Funktion aktiviert Facebook Damit können Sie a senden wohnen Veranstaltung über Ihre Unternehmensseite oder Ihr persönliches Profil für Ihr Publikum. Live-Videos werden während des. Im Newsfeed einer Person angezeigt Live-Event sowie nach dem Veranstaltung ist beendet.

Hinweis: Ausführliche Informationen zum Live-Streaming von Facebook finden Sie in der Facebook-Dokumentation.

Führen Sie die folgenden Schritte aus, um den Live-Stream in Facebook zu verwenden, nachdem das Live-Streaming im Live-Modul gestartet wurde.

  1. Melden Sie sich bei Ihrem Facebook-Konto an.
  2. Erstellen Sie einen neuen Beitrag und klicken Sie auf Live-Video Möglichkeit.
  3. Facebook fordert Sie möglicherweise auf, auf Ihre Computerkamera und Ihr Mikrofon zuzugreifen. In diesem Thema wird eine externe DSLR-Kamera für die Veranstaltung verwendet. Klicken Sie also auf Verbinden.
  4. Facebook zeigt a Server-URL und Stream-Schlüssel. Diese Werte werden im Live-Modul verwendet.
  5. Kopiere das Server-URL in die Zwischenablage.
  6. Fügen Sie im Live-Modul die ein Server-URL in die RTMP-Ausgabe-URL Feld auf der Seite Kontrollraum.
  7. Kopiere das Stream-Schlüssel von Facebook in die Zwischenablage.
  8. Fügen Sie im Live-Modul die ein Stream-Schlüssel am Ende von RTMP-Ausgabe-URL Feld. Das RTMP-Ausgabe-URL Feld wird das Facebook sein Server-URL mit dem Stream-Schlüssel angehängt, alle als ein Wert.
    Beachten Sie das nur aktuelles Format für RTMP-URLs unterstützt ist rtmps://host:port/application/streamName.
  9. Wählen Sie eine Wiedergabe aus Wählen Sie Rendition Dropdown und klicken RTMP-Ausgabe hinzufügen.
  10. Vergewissern Sie sich unterhalb des RTMP-Abschnitts, dass die Verbindung erfolgreich war.
  11. Bestätigen Sie, dass der Live-Stream in Facebook angezeigt wird. Geben Sie einen Titel und eine Beschreibung für Facebook ein.
  12. Klicken Geh Leben.

Der Live-Stream beginnt in Facebook.

Klicken Fertig um den Facebook Live Stream zu beenden. Denken Sie daran, den Encoder auch zu stoppen, wenn das Ereignis abgeschlossen ist.

Streaming auf YouTube

YouTube bietet die Möglichkeit, Live-Streaming-Videos für Ihr Publikum bereitzustellen. Bevor du zum ersten Mal einen Live-Stream startest, musst du deinen YouTube-Kanal aktivieren.

Hinweis: Aktivieren Ein Live-Stream zum ersten Mal kann bis zu 24 Stunden dauern. Einmal aktiviert, kann dein Stream sofort live gehen.
Hinweis: Ausführliche Informationen zum YouTube-Live-Streaming finden Sie in der YouTube-Dokumentation.

Nachdem das Live-Streaming im Live-Modul gestartet wurde, führen Sie die folgenden Schritte aus, um den Live-Stream in YouTube zu verwenden.

  1. Melde dich bei deinem YouTube-Konto an.
  2. Gehe zum YouTube Studio Seite unter https://studio.youtube.com.
  3. Klicken Sie in der linken Navigation auf Weitere Funktionen> Live-Stream jetzt.
  4. YouTube fordert Sie möglicherweise auf, auf Ihre Computerkamera und Ihr Mikrofon zuzugreifen. In diesem Thema wird eine externe DSLR-Kamera für die Veranstaltung verwendet. Klicken Sie also auf Verbinden.
  5. YouTube zeigt a Server-URL und Stream Name / Schlüssel. Diese Werte werden im Live-Modul verwendet.
  6. Kopiere das Server-URL in die Zwischenablage.
  7. Fügen Sie im Live-Modul die ein Server-URL in die RTMP-Ausgabe-URL Feld auf der Seite Kontrollraum.
    Beachten Sie das nur aktuelles Format für RTMP-URLs unterstützt ist rtmps://host:port/application/streamName.
  8. Kopiere das Stream Name / Schlüssel von YouTube in die Zwischenablage.
  9. Fügen Sie im Live-Modul die ein Stream Name / Schlüssel am Ende von RTMP-Ausgabe-URL Feld. Das RTMP-Ausgabe-URL Feld wird das YouTube sein Server-URL mit dem Stream Name / Schlüssel angehängt, alle als ein Wert.
  10. Wählen Sie eine Wiedergabe aus Wählen Sie Rendition Dropdown und klicken RTMP-Ausgabe hinzufügen.
  11. Vergewissern Sie sich unterhalb des RTMP-Abschnitts, dass die Verbindung erfolgreich war.
  12. Bestätige, dass der Live-Stream in YouTube angezeigt wird.

Der Live-Stream beginnt in YouTube.

Klicken Kehre zu YouTube Studio zurück und stoppen Sie dann den Encoder, um den Live-Stream zu beenden. Wenn Ihr Stream fertig ist, wird automatisch ein öffentliches Video erstellt und hochgeladen, damit die Benutzer es später ansehen können.
