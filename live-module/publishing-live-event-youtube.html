Veröffentlichen eines Live-Events auf YouTube

In diesem Thema erfahren Sie, wie Sie mit Brightcove Social ein Live-Ereignis auf YouTube veröffentlichen.

Mit der Funktion "Brightcove Live to Social Media" können Sie ein Live-Ereignis aus dem Social-Modul heraus planen und anschließend an YouTube senden. Dazu werden folgende Aufgaben erledigt:

  • Planen Sie ein YouTube-Live-Event
  • Erstellen Sie ein Live-Ereignis und weisen Sie es einem geplanten sozialen Ereignis zu
  • Vorschau und Übertragung des Live-Streams
  • Beenden Sie den Live-Stream

Planen eines YouTube Live-Ereignisses

Wenn Sie ein YouTube-Live-Event im Voraus planen, können Sie das Event auf Ihrem YouTube veröffentlichen Publikum vor Beginn der Veranstaltung. Bevor ein Live-Ereignis geplant werden kann, muss ein Veröffentlichungsziel für die Youtube-Site erstellt werden, auf die Sie das Ereignis streamen möchten. Informationen zum Erstellen von Veröffentlichungszielen finden Sie unter Erstellen von Social Media Publishing-Zielen.

Um einen Livestream auf YouTube zu erstellen, musst du verifizieren Sie Ihr Konto Aktivieren Sie dann Live-Streaming für Ihr Konto. Dieser Schritt kann bis zu 24 Stunden dauern, bis er aktiviert ist.

Befolgen Sie diese Schritte, um ein bevorstehendes YouTube-Live-Video zu planen.

  1. Öffnen Sie das Modul Soziales.
  2. Von dem Veröffentlicht Seite, klicken Sie Live-Event erstellen.
  3. Wählen Sie das Ziel aus, zu dem das Ereignis gestreamt wird, und klicken Sie dann auf Weiter.
  4. Geben Sie auf der Seite Live-Ereignis planen optional einen ein Titel, Beschreibung und wählen Sie a YouTube-Kategorie. Diese Werte werden in deinem Youtube-Beitrag angezeigt und legen den Live-Stream fest.
  5. In dem ZEITPLAN Verwenden Sie die Kalender- und Uhrzeitsteuerung, um die Einstellungen vorzunehmen Datum und Uhrzeit des Ereignisses Für das Event.
  6. Wählen Sie das entsprechende aus Zeitzone.
  7. In dem LIVE EINSTELLUNGEN Abschnitt, konfigurieren Sie die YouTube Live Video-Einstellungen wie angemessen.
    • DVR aktivieren oder deaktivieren - Wenn diese Option aktiviert ist, können Ihre Zuschauer während des Ereignisses pausieren, zurückspulen und fortfahren. Sobald ein Zuschauer die Wiedergabe fortsetzt, wird das Ereignis dort fortgesetzt, wo er die Pause gedrückt hat.
    • Sichtweite - Mit dieser Option können Sie die Datenschutzeinstellungen Ihres Live-Videos auswählen.
    • Lizenz - Mit dieser Option können Sie die Lizenz und die Rechte des Videos auswählen.
  8. Drücke den Benutzerdefiniertes Bild hinzufügen Link oben auf der Seite.
  9. Klicken Bild auswählen um ein Bild zu durchsuchen und auszuwählen oder ein zu verwendendes Bild per Drag & Drop zu verschieben. Dieses Bild wird im YouTube-Beitrag angezeigt, in dem das Live-Event angekündigt wird. Das Vorschaubild wird aktualisiert, um das neue Bild anzuzeigen.
  10. Klicken OK habe es.
  11. Klicken Ereignis planen um die Veranstaltung in YouTube zu planen.
  12. Drücke den Auf YouTube ansehen Link oben auf der Seite, um den Beitrag auf YouTube anzuzeigen.

Erstellen eines Live-Ereignisses

Bevor Sie mit dem Streaming eines Live-Ereignisses beginnen können, muss mit dem Live-Modul ein Live-Ereignis erstellt werden. Im Live-Modul erstellte Live-Ereignisse können über das Social-Modul übertragen werden.

Beachten Sie beim Erstellen eines Ereignisses im Live-Modul die folgenden Punkte:

  • Wenn du ein wiederkehrendes Ereignis erstellst, kannst du das Ereignis vorab erstellen und aktivieren, wenn du bereit bist, mit dem Streaming zu Youtube zu beginnen.
  • Wenn Sie ein reguläres Ereignis erstellen (einmalig), muss das Live-Ereignis innerhalb von 30 Minuten nach dem Erstellen des Ereignisses gestartet werden. Wenn du ein reguläres Event verwendest, sollte es erstellt werden, bevor du bereit bist, auf YouTube live zu gehen.

Ausführliche Informationen zum Erstellen von Live-Ereignissen mit dem Live-Modul finden Sie unter Erstellen und Verwalten von Live-Ereignissen mit dem Live-Modul.

Führen Sie die folgenden Schritte aus, um ein neues Live-Ereignis zu erstellen.

  1. Öffnen Sie das Live-Modul.
  2. Klicken Ereignis erstellen. Die Seite Live-Ereignis erstellen wird geöffnet.
  3. Geben Sie die Veranstaltungsname und alle erforderlichen Felder.
  4. Für die Streaming zu Social Media Option auswählen Dürfen.
  5. Wählen Sie a Live-Aufnahmeprofil.
  6. Klicken ERWEITERTE OPTIONEN und wählen Sie die Region am nächsten an der Stelle, an der sich Ihr Encoder befindet.
  7. Wenn Sie ein wiederkehrendes Ereignis erstellen möchten, wählen Sie die Option Erstellen Sie dies als wiederkehrendes Ereignis mithilfe eines statischen Einstiegspunkts (SEP). Möglichkeit.
  8. Klicken Ereignis erstellen. Die Seite Kontrollraum wird geöffnet und a angezeigt Streaming-Endpunkt (RTMP-URL) und Stream Name die wir von Ihrem Encoder verwenden werden.
  9. Klicken Sie auf, um ein Live-Ereignis mithilfe des Live-Moduls mit einem geplanten sozialen Ereignis zu verbinden Stellen Sie eine Verbindung zu Social Media her in der linken Navigation. Beachten Sie, dass das Live-Event auch über das zugewiesen werden kann Soziales Modul.
  10. Klicken Sie auf ein Ereignis und dann auf Verbinden.
  11. Klicken Sie in der linken Navigation auf den Link Kontrollraum, um zum Kontrollraum zurückzukehren.

Das Streaming-Endpunkt (RTMP-URL) und Stream Name Werte werden von Ihrem Encoder verwendet. Starten Sie den Live-Stream und bestätigen Sie, dass er im Kontrollraum angezeigt wird.

Zuweisen eines Live-Ereignisses zu einem geplanten sozialen Ereignis

Wenn das Live-Ereignis mit dem Live-Modul keinem geplanten Ereignis zugewiesen wurde, muss das zu verwendende Live-Ereignis im Modul "Sozial" ausgewählt werden. Führen Sie die folgenden Schritte aus, um einem geplanten Ereignis mithilfe des Moduls "Sozial" ein Live-Ereignis zuzuweisen.

Diese Schritte sind nur erforderlich, wenn das Live-Ereignis nicht im Live-Modul zugewiesen wurde.
  1. Kehren Sie zum Modul "Soziales" zurück und bearbeiten Sie das geplante Ereignis.
  2. Drücke den Videoquelle anschließen Link oben auf der Seite.
  3. Wählen Sie das Live-Event aus Wählen Sie Videoquelle Dropdown-Liste.
  4. Klicken Quelle verbinden.

Vorschau und Übertragung des Live-Streams

Vor der Vorschau des Ereignisses im Modul "Sozial" sollte das Live-Ereignis gestartet und im Live-Modul sichtbar sein.

Führen Sie die folgenden Schritte aus, um eine Vorschau des Live-Streams anzuzeigen und diesen zu übertragen.

  1. Wenn das Ereignis mithilfe des Live-Moduls zugewiesen wurde, kehren Sie zum Modul "Soziales" zurück und bearbeiten Sie das geplante Ereignis.
  2. Bestätigen Sie, dass die Stream-Vorschau angezeigt wird. Dies ist eine Vorschau aus dem Live-Modul.
  3. Drücke den Vorschau des Live-Streams Link oben auf der Seite und klicken Sie dann auf Vorschau. Drücke den abspielen Schaltfläche, um eine Vorschau des YouTube-Live-Streams anzuzeigen. Dies ist der Stream, den YouTube sendet. Bestätige also, dass er wie erwartet ist.
  4. Um den Live-Stream auf Youtube zu übertragen, klicken Sie auf Sendung an Zuschauer Link und dann klicken Broadcast-Stream. Die Vorschau wird aktualisiert und zeigt die Anzahl der Zuschauer an.

Drücke den Auf YouTube ansehen Link unter dem Player, um den YouTube-Player anzuzeigen.

Der Live-Stream wird auf Ihrer Youtube-Seite angezeigt.

Wenn YouTube-Kommentare von Zuschauern hinzugefügt werden, können sie im Social-Modul angezeigt werden, indem Sie auf klicken Kommentar Feed Tab.

Beenden des Live-Streams

Führen Sie die folgenden Schritte aus, um den Live-Stream zu beenden.

  1. Kehren Sie zum Modul "Soziales" zurück und bearbeiten Sie Ihre geplante Veranstaltung.
  2. Drücke den Social Live Event beenden Link oben auf der Seite und klicken Sie dann auf Live-Event beenden.
  3. Kehren Sie zum Live-Modul zurück, um das Ereignis zu beenden und den Encoder anzuhalten.
  4. Wenn Sie ein wiederkehrendes Ereignis verwendet haben, kehren Sie zum Live-Modul zurück und deaktivieren Sie das Ereignis.
Stellen Sie sicher, dass der Encoder geschlossen und gestoppt ist, wenn der Livestream abgeschlossen ist. Wenn eine Verbindung zwischen einem Encoder und Brightcove geöffnet wird, kann es zu unerwarteten Verbindungsgebühren führen, selbst wenn keine Daten gesendet werden.

Abgeschlossene Ereignisse werden im Modul "Soziales" auf der Seite "Veröffentlicht" angezeigt. Klicken Sie auf das Löschen-Symbol ( ) entfernt die Veranstaltung von YouTube.
