Veröffentlichen eines Live-Events auf Facebook

{{ page.description }}
 

Mit der Funktion "Brightcove Live to Social Media" können Sie ein Live-Ereignis innerhalb des Social-Moduls planen und anschließend an Facebook senden. Dazu werden folgende Aufgaben erledigt:

  • Planen Sie ein Facebook-Live-Event
  • Erstellen Sie ein Live-Ereignis und weisen Sie es einem geplanten sozialen Ereignis zu
  • Vorschau und Übertragung des Live-Streams
  • Beenden Sie den Live-Stream

Planen eines Facebook-Live-Events

Wenn Sie ein Facebook-Live-Event im Voraus planen, können Sie das Event vor Beginn des Events Ihrem Facebook-Publikum bekannt machen. Bevor ein Live-Ereignis geplant werden kann, muss ein Veröffentlichungsziel für die Facebook-Site erstellt werden, auf die Sie das Ereignis streamen möchten. Informationen zum Erstellen von Veröffentlichungszielen finden Sie unter Erstellen von Social Media Publishing-Zielen.

Befolgen Sie diese Schritte, um ein bevorstehendes Facebook-Live-Video zu planen.

  1. Öffnen Sie das Modul Soziales.
  2. Von dem Veröffentlicht Seite, klicken Sie Live-Event erstellen.
  3. Wählen Sie das Ziel aus, zu dem das Ereignis gestreamt wird, und klicken Sie dann auf Weiter.
  4. Geben Sie auf der Seite Live-Ereignis planen optional einen ein Titel und Statusnachricht. Diese Werte werden im Facebook-Beitrag angezeigt, in dem der Live-Stream angekündigt wird.
  5. In dem ZEITPLAN Verwenden Sie die Kalender- und Uhrzeitsteuerung, um die Einstellungen vorzunehmen Datum und Uhrzeit des Ereignisses Für das Event.
  6. Wählen Sie das entsprechende aus Zeitzone.
  7. In dem LIVE EINSTELLUNGEN Abschnitt, konfigurieren Sie die Facebook Live Video Einstellungen wie angemessen.
    • Einbettung zulassen - Wenn diese Option aktiviert ist, kann das Video auf Websites von Drittanbietern eingebettet werden
    • Nach Veröffentlichung des Live-Videos nicht mehr veröffentlichen - Wenn diese Option aktiviert ist, wird das Video nach Beendigung des Ereignisses nicht veröffentlicht (für die Öffentlichkeit nicht sichtbar)
  8. Konfigurieren Sie die ZIEL und RECHTE Abschnitte nach Bedarf. Beachten Sie, dass der Abschnitt RECHTE nur angezeigt wird, wenn das verbundene Konto Zugriff auf das hat Facebook Rights Manager.
  9. Drücke den Benutzerdefiniertes Bild hinzufügen Link oben auf der Seite.
  10. Klicken Bild auswählen um ein Bild zu durchsuchen und auszuwählen oder ein zu verwendendes Bild per Drag & Drop zu verschieben. Dieses Bild erscheint in dem Facebook-Beitrag, in dem das Live-Event angekündigt wird. Das Vorschaubild wird aktualisiert, um das neue Bild anzuzeigen.
  11. Klicken OK habe es.
  12. Klicken Ereignis planen um die Veranstaltung in Facebook zu planen.
  13. Drücke den Auf Facebook ansehen Link oben auf der Seite, um den Beitrag auf Facebook anzuzeigen.

Erstellen eines Live-Events

Bevor Sie mit dem Streaming eines Live-Ereignisses beginnen können, muss mit dem Live-Modul ein Live-Ereignis erstellt werden. Im Live-Modul erstellte Live-Ereignisse können über das Social-Modul übertragen werden.

Beachten Sie beim Erstellen eines Ereignisses im Live-Modul die folgenden Punkte:

  • Wenn Sie ein wiederkehrendes Ereignis erstellen, können Sie das Ereignis im Voraus erstellen und aktivieren, wenn Sie bereit sind, mit dem Streaming zu Facebook zu beginnen.
  • Wenn Sie ein reguläres Ereignis erstellen (einmalig), muss das Live-Ereignis innerhalb von 30 Minuten nach dem Erstellen des Ereignisses gestartet werden. Wenn Sie eine reguläre Veranstaltung verwenden, sollte diese erstellt werden, bevor Sie bereit sind, auf Facebook live zu gehen.

Ausführliche Informationen zum Erstellen von Live-Ereignissen mit dem Live-Modul finden Sie unter Erstellen und Verwalten von Live-Ereignissen mit dem Live-Modul.

Führen Sie die folgenden Schritte aus, um ein neues Live-Ereignis zu erstellen.

  1. Öffnen Sie das Live-Modul.
  2. Klicken Ereignis erstellen. Die Seite Live-Ereignis erstellen wird geöffnet.
  3. Geben Sie die Veranstaltungsname und alle erforderlichen Felder.
  4. Für die Streaming zu Social Media Option auswählen Dürfen.
  5. Wählen Sie a Live-Aufnahmeprofil.
  6. Klicken ERWEITERTE OPTIONEN und wählen Sie die Region am nächsten an der Stelle, an der sich Ihr Encoder befindet.
  7. Wenn Sie ein wiederkehrendes Ereignis erstellen möchten, wählen Sie die Option Erstellen Sie dies als wiederkehrendes Ereignis mithilfe eines statischen Einstiegspunkts (SEP). Möglichkeit.
  8. Klicken Ereignis erstellen. Die Seite Kontrollraum wird geöffnet und a angezeigt Streaming-Endpunkt (RTMP-URL) und Stream Name die wir von Ihrem Encoder verwenden werden.
  9. Klicken Sie auf, um ein Live-Ereignis mithilfe des Live-Moduls mit einem geplanten sozialen Ereignis zu verbinden Stellen Sie eine Verbindung zu Social Media her in der linken Navigation. Beachten Sie, dass das Live-Event auch über das zugewiesen werden kann Soziales Modul.
  10. Klicken Sie auf ein Ereignis und dann auf Verbinden.
  11. Klicken Sie in der linken Navigation auf den Link Kontrollraum, um zum Kontrollraum zurückzukehren.

Das Streaming-Endpunkt (RTMP-URL) und Stream Name Werte werden von Ihrem Encoder verwendet. Starten Sie den Live-Stream und bestätigen Sie, dass er im Kontrollraum angezeigt wird.

Zuweisen eines Live-Ereignisses zu einem geplanten sozialen Ereignis

Wenn das Live-Ereignis mit dem Live-Modul keinem geplanten Ereignis zugewiesen wurde, muss das zu verwendende Live-Ereignis im Modul "Sozial" ausgewählt werden. Führen Sie die folgenden Schritte aus, um einem geplanten Ereignis mithilfe des Moduls "Sozial" ein Live-Ereignis zuzuweisen.

Hinweis: Diese Schritte sind nur erforderlich, wenn das Live-Event nicht im Live-Modul zugewiesen wurde.
  1. Kehren Sie zum Modul "Soziales" zurück und bearbeiten Sie das geplante Ereignis.
  2. Drücke den Videoquelle anschließen Link oben auf der Seite.
  3. Wählen Sie das Live-Event aus Wählen Sie Videoquelle Dropdown-Liste.
  4. Klicken Quelle verbinden.

Vorschau und Übertragung des Live-Streams

Vor der Vorschau des Ereignisses im Modul "Sozial" sollte das Live-Ereignis gestartet und im Live-Modul sichtbar sein.

Führen Sie die folgenden Schritte aus, um eine Vorschau des Live-Streams anzuzeigen und diesen zu übertragen.

  1. Wenn das Ereignis mithilfe des Live-Moduls zugewiesen wurde, kehren Sie zum Modul "Soziales" zurück und bearbeiten Sie das geplante Ereignis.
  2. Bestätigen Sie, dass die Stream-Vorschau angezeigt wird. Dies ist eine Vorschau aus dem Live-Modul.
  3. Drücke den Vorschau des Live-Streams Link oben auf der Seite und klicken Sie dann auf Vorschau. Drücke den abspielen Schaltfläche, um eine Vorschau des Facebook-Live-Streams anzuzeigen. Dies ist der Stream, den Facebook sendet. Bestätigen Sie also, dass er wie erwartet ist.
  4. Klicken Sie auf, um den Live-Stream an Facebook zu senden Sendung an Zuschauer Link und dann klicken Broadcast-Stream. Die Vorschau wird aktualisiert und zeigt die Anzahl der Zuschauer an.

Drücke den Wechseln Sie zum Facebook-Player Link unter dem Player, um den Facebook-Player anzuzeigen.

Der Live-Stream kann auch auf Ihrer Facebook-Seite angezeigt werden.

Wenn Facebook-Kommentare von Zuschauern hinzugefügt werden, können sie innerhalb des Social-Moduls angezeigt werden, indem Sie auf klicken Kommentar Feed Tab.

Beenden des Live-Streams

Führen Sie die folgenden Schritte aus, um den Live-Stream zu beenden.

  1. Kehren Sie zum Modul "Soziales" zurück und bearbeiten Sie Ihre geplante Veranstaltung.
  2. Drücke den Social Live Event beenden Link oben auf der Seite und klicken Sie dann auf Live-Event beenden.
  3. Kehren Sie zum Live-Modul zurück, um das Ereignis zu beenden und den Encoder anzuhalten.
  4. Wenn Sie ein wiederkehrendes Ereignis verwendet haben, kehren Sie zum Live-Modul zurück und deaktivieren Sie das Ereignis.
Hinweis: Stellen Sie sicher, dass der Encoder geschlossen / gestoppt ist, wenn der Live-Stream abgeschlossen ist. Wenn eine Verbindung zwischen einem Encoder und Brightcove geöffnet wird, kann es zu unerwarteten Verbindungsgebühren führen, selbst wenn keine Daten gesendet werden.

Abgeschlossene Ereignisse werden im Modul "Soziales" auf der Seite "Veröffentlicht" angezeigt. Klicken Sie auf das Löschen-Symbol ( ) wird die Veranstaltung von Facebook entfernen.
