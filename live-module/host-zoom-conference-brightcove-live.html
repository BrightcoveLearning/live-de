Einführung

Zoom-Videokommunikation ist ein Technologieunternehmen, das eine Cloud-basierte Peer-to-Peer-Softwareplattform für Telekonferenzen, Telearbeit und Fernunterricht anbietet. Um ein größeres Publikum zu erreichen, können Sie Brightcove Live nutzen, um Ihr Meeting für andere zu streamen.

Schritte

Führen Sie die folgenden Schritte aus, um eine Konferenz oder ein Webinar für Brightcove Live abzuhalten:

  1. Konferenzen

    Gehen Sie für Zoom-Konferenzen wie folgt vor:

    1. Navigieren Sie zu Ihrem Zoom-Meeting die Einstellungen und scrollen Sie zu In Besprechung (Erweitert) Sektion.
    2. Aktivieren Sie die Live-Streaming-Meetings zulassen Option durch Klicken auf den Kippschalter. Wenn diese Option nicht angezeigt wird, wenden Sie sich an Ihren Zoom Account Manager, um zu erfahren, wie Sie diese Funktion Ihrem Konto hinzufügen können.

    3. Wählen Benutzerdefinierter Live-Streaming-Dienst und geben Sie optional Textnotizen in das Feld ein.

    Webinare

    Gehen Sie für Zoom-Webinare wie folgt vor:

    1. Navigieren Sie zu unseren Zoom-Webinar-Einstellungen und scrollen Sie zu In den Webinar-Einstellungen Sektion.
    2. Aktivieren Sie die Ermöglichen Sie Hosts, ihre Webinare live zu streamen Möglichkeit.

    3. Wählen Sie Benutzerdefinierter Streaming-Dienst aus und speichern Sie.

  2. Erstellen Sie in Video Cloud Studio eine Live-Event mit dem Medienmodul.

    Wenn Sie die Übertragung nach Abschluss speichern möchten, überprüfen Sie die Event in Video-Asset umwandeln, wenn abgeschlossen Checkbox!

  3. Wenn Sie ein Live-Ereignis erstellen, haben Sie Zugriff auf die Streaming-URL und den Streaming-Schlüssel.

    • Streaming-URL - Streaming-Endpunkt-URL (Sie finden die Endpunkt-URL im Brightcove Live Control Room
    • Streaming Key - Stream Name (In Brightcove lautet der Stream-Name immer "am Leben")
  4. Veröffentlichen und binden Sie den Player für dieses Live-Ereignis in Ihre Website / Intranetseite ein und notieren Sie sich die Seiten-URL.
  5. Konferenzen

    Navigieren Sie für Zoom-Konferenzen zu Ihrem geplantes Treffen Listen Sie das Meeting auf, das Sie streamen möchten, und wählen Sie es aus.

    Webinare

    Kehren Sie für Zoom-Webinare zu Bildschirm "Webinare" und planen Sie ein Webinar. Wähle aus Live-Streaming Tab.

  6. Scrollen Sie zu Live-Streaming Sektion
  7. Wähle aus Konfigurieren Sie die Live-Stream-Einstellungen Verknüpfung. Hier geben Sie die Streaming-URL, den Streaming-Schlüssel und die URL der Seite ein, auf der Sie den Player hosten.
  8. Wählen Sie in Ihrem Zoom-Aufruf die Option aus MEHR Schaltfläche und wählen Sie die Live auf benutzerdefinierten Live-Streaming-Service Möglichkeit. Dadurch wird der Live-Stream gestartet.
