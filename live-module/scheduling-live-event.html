--- title: Planung von Live-SEP-Events parent: Live Module ---

Automatischer Start/Stopp von Live-Events mit statischen Einstiegspunkten

Mit dieser Funktion kann der Benutzer die Start- und Stoppzeit eines oder mehrerer Live Job-Ereignisse planen und verwalten.

Voraussetzungen

Um Ereignisse zu planen, muss ein Live-Ereignis vom Typ SEP (Static Entry Point) erstellt werden.

Derzeit wird das Scheduling von redundanten Gruppen nicht unterstützt.

Empfehlungen und Best Practices

Für beste Ergebnisse folgen Sie bitte den Anweisungen unten.

  1. Es können jeweils sechs ausstehende Aktivierungen geplant werden. Wenn weitere benötigt werden, wenden Sie sich bitte an den Kundendienst.
  2. Sobald die SEP aktiviert ist, ist die standardmäßige maximale Wartezeit weiterhin gültig (siehe Live-API: Statische Einstiegspunkte für weitere Informationen).
  3. Beachten Sie, dass die in der Statische Einstiegspunkte sind noch gültig.

Planungsfenster im Kontrollraum

Wenn ein Live-Konto für Auto Start / Stop aktiviert ist, ist in der oberen rechten Ecke des Kontrollraums unter dem Preview-Player ein Planungsfenster verfügbar.

Befolgen Sie die nachstehenden Anweisungen, um Auto Start / Stop zu verwenden:

  1. Erstellen Sie im Live-Modul ein neues Live-Ereignis mit einem statischen Einstiegspunkt oder besuchen Sie ein zuvor erstelltes.
  2. In dem Kontrollraum wird ein neues Panel mit dem Namen Scheduling verfügbar sein. Klicken + Zeitplan hinzufügen um eine neue automatische Aktivierung und Deaktivierung zu erstellen.
  3. Ändern Sie das Start- und Enddatum auf die gewünschte Aktivierung bzw. Deaktivierung.
    Obwohl wir nicht verhindern, dass mehrere Zeitpläne gleichzeitig ausgeführt werden, werden die in Konflikt stehenden Zeilen mit einer Warnung angezeigt. Widersprüchliche Zeitpläne schlagen wahrscheinlich fehl, wenn sie nicht vor der Aktivierungszeit behoben werden. Um die Warnung zu vermeiden, löschen Sie einfach eine der Zeilen oder passen Sie die Zeiten an.
  4. Um einen zweiten Zeitplan zu erstellen, klicken Sie erneut auf + Zeitplan hinzufügen Taste. Das + Zeitplan hinzufügen Die Schaltfläche wird automatisch deaktiviert, wenn das Limit der ausstehenden Aktivierungen erreicht ist.
  5. Um einen Zeitplan zu löschen, klicken Sie auf das -Symbol.
  • Zeitplanereignisse werden als Warnung gelb, wenn widersprüchliche Zeitpläne vorliegen.
  • Geplante Ereignisse werden rot, wenn das Streaming aus irgendeinem Grund fehlschlägt.
  • Die fehlgeschlagenen Zeitpläne bleiben 7 Tage lang sichtbar.

Anstehende Aktivierungen anzeigen

  1. Sobald geplante Aktivierungen erstellt wurden, können Sie sie im Ereignisse verwalten> Stehen zu Tab. Das nächste Mal Start und Halt Daten werden unter dem angezeigt Bevorstehende Titel.
  2. Das Show Mit dem Auswahlfeld können Sie nur geplante Ereignisse anzeigen.
  3. Sobald ein Ereignis beginnt, wird es auf dem angezeigt Wohnen Registerkarte und die Start Zeit wird deaktiviert.
