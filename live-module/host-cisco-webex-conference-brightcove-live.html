Einführung

Cisco Webex ist eine Videokonferenzanwendung, mit der Sie sich online mit anderen Personen treffen können. Um ein größeres Publikum zu erreichen, können Sie Brightcove Live nutzen, um Ihr Meeting für andere zu streamen.

Schritte

Führen Sie die folgenden Schritte aus, um ein Meeting für Brightcove Live auszurichten:

  1. Melden Sie sich als Site-Administrator bei Ihrer Cisco Webex-Plattform an.

  2. Aktivieren Sie mit der Site-Administration oder dem Control Hub den Live-Stream-Zugriff auf andere Streaming-Dienste.

  3. Aktivieren Sie mit der Site-Administration oder dem Control Hub Live-Streaming für einzelne Benutzer.

    Navigieren Sie zu Benutzer und wählen Sie ein Benutzerkonto aus, das aktualisiert werden soll. Klicken Erweiterte Benutzereinstellungen und Tracking-Codes Abschnitt für den Benutzer, und überprüfen Sie die Aktivieren Sie das Live-Streaming Box.

  4. Einloggen in Video Cloud Studio von Brightcove.

  5. Navigieren Sie zum Live-Modul und erstellen Sie ein Brightcove Live-Ereignis. Einzelheiten finden Sie in der Erstellen und Verwalten von Live-Ereignissen mit dem Live-Modul Dokument.

    Wenn Sie die Übertragung nach Abschluss speichern möchten, überprüfen Sie die Event in Video-Asset umwandeln, wenn abgeschlossen Checkbox!

  6. Wenn Sie dieses Ereignis erstellen, erhalten Sie die folgenden Informationen:

    • Endpunkt (RTMP-URL) - Streaming-URL
    • Stream Name - Streaming-Schlüssel, wobei der Wert immer ist am Leben
  7. Veröffentlichen Sie den Player für dieses Live-Event und binden Sie ihn in Ihre Website oder Intranetseite ein. Kopieren Sie die Seiten-URL in Ihre Zwischenablage.

  8. Kehren Sie zur Cisco Webex-Anwendung zurück. Starten oder planen Sie Ihr Webex-Meeting. Wählen Sie dazu das vertikale 3-Punkt-Symbol aus und wählen Sie Starten Sie das Live-Streaming.

  9. In dem Weitere Informationen zum Streaming-Dienst Geben Sie im Dialogfeld Folgendes ein:

    • Streaming-Dienst:: Benutzen Brightcove
    • Ziel-Stream-Link:: Verwenden Sie die Endpunkt (RTMP-URL) Wert von Brightcove Live.
    • Ziel-Stream-Schlüssel:: Benutzen alive (Es ist immer dieser Wert)

  10. Das ist es. Du lebst!
