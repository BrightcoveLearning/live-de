Unterstützte Encoder für Live-Events

Dieses Thema soll einige Richtlinien für Hardware-Encoder-Einstellungen bei der Durchführung eines Live-Stream-Ereignisses enthalten. Eine Liste aller von Brightcove verfügbaren Optionen für die Bereitstellung von Live-Streaming-Videos finden Sie unter Live-Streams bereitstellen.

Im Folgenden finden Sie eine Liste von Encodern, von denen allgemein bekannt ist, dass sie mit Video Cloud funktionieren. Andere Encoder können auch funktionieren; diese Liste enthält die Encoder, die wir speziell getestet haben.

Beachten Sie auch, dass unterstützt bedeutet nicht, dass alle Funktionen unterstützt werden. Bestimmte Funktionen (z. B. das Senden von Cuepoints) unterscheiden sich zwischen verschiedenen Encodern.
  • xSplit
  • OBS
  • Wirecast
  • Cambria Live
  • Liveshell
  • AWS Elementar Live
  • Haivision
  • LiveU
  • Matrox
  • NewTek
  • Teradek
