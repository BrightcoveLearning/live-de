Mehrsprachiges Audio

Mit dieser Funktion können mehrere Audiosprachen, beschreibende oder Umgebungs-Audioerlebnisspuren für die Live-Wiedergabe aufgenommen werden.

Voraussetzungen:

  • Definiere das PID (Packet Identifier) für jede Audiospur in Ihrem Encoder.
  • Derzeit wird nur das RTP-Eingabeformat unterstützt. Damit dieser Abschnitt in der aktiviert wird Live-Event erstellen Seite, gehen Sie zur ERWEITERTE OPTIONEN Abschnitt und wählen Sie Echtzeit-Transportprotokoll (RTP) unter dem Eingabeformat Wähler.
  • In dem CIDR Whitelist Abschnitt, geben Sie die IP ein: 0.0.0.0/0 um eine Verbindung zum RTP-Endpunkt herzustellen. Die Schaltfläche Mehrsprachig wird deaktiviert, wenn das Eingabeformat nicht auf RTP eingestellt ist.
    • Anmerkungen:
    • Informationen zum Definieren des PID (Packet Identifier) Informationen zu den Audiospuren finden Sie in Ihrer Encoder-Dokumentation.
    • Unterstützung für SRT wird in naher Zukunft hinzugefügt.
    • Weitere Informationen zum Einrichten eines RTP-Eingangs finden Sie unter Konfigurieren der Dokumentation zu erweiterten Optionen.

Hinzufügen von Audiospuren zu einem Live-Event

Beim Erstellen eines neuen Live-Ereignisses:

  1. Öffne das Audiospuren Sektion.
  2. Drücke den Fügen Sie Audiospuren hinzu Taste.
  3. Füllen Sie die Felder aus, um eine neue Audiospur hinzuzufügen. Felder mit einem Sternchen (*) sind erforderlich.
    • Standard:: Wählen Sie die Audiospur als Standard aus.
    • Sprache*:: Geben Sie den Sprachcode der neuen Audiospur ein. Beispiel: en, es, cy-GB usw.
    • PID *:: Geben Sie den zuvor in Ihrem Encoder definierten PID-Code ein, um die Audiospuren in Ihrem eingehenden Stream mit der richtigen Sprache und Variante zu synchronisieren. Hinweis: Für jede Audiospur sind eindeutige PIDs erforderlich.
    • Variante:: Repräsentiert die Variante der Audiospur. Dies können sein: Haupt, Kommentar, Alternative, Bildunterschrift, Untertitel, Ergänzung, Synchronisation.
    • Etikette:: Geben Sie einen Namen für die Spur ein. Beispiel: Englisch, Spanisch usw. Beachten Sie, dass der Sprachcode als Beschriftung verwendet wird, wenn das Beschriftungsfeld leer bleibt.
  4. Durch Anklicken des + und - Schaltflächen können Sie Audiospuren hinzufügen oder entfernen.
    Hinweis: Sie können bis zu sechs Audiospuren hinzufügen. Wenn Sie weitere Tracks hinzufügen müssen, wenden Sie sich bitte an den Support.

Ändern der Audiospur

  1. Wenn die Multi-Audio-Spuren korrekt eingerichtet wurden, verfügt der Player über ein Kopfhörersymbol, mit dem Sie zwischen den Spuren wechseln können.
