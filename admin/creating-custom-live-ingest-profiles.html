Erstellen von benutzerdefinierten Live-Aufnahmeprofilen

In diesem Thema erfahren Sie, wie Sie benutzerdefinierte Live-Aufnahmeprofile erstellen und verwalten.

Auf der Seite Verwaltung von Aufnahmeprofilen werden benutzerdefinierte Live-Aufnahmeprofile verwaltet. Live-Aufnahmeprofile bestimmen die Anzahl und Qualität der von Brightcove Live erstellten Wiedergaben. Die Live-Aufnahmeprofile werden beim Hochladen von Videos in ein Video Cloud-Konto nicht verwendet. Aufnahmeprofile werden mit JSON definiert. Überprüfen Sie das, bevor Sie ein neues Aufnahmeprofil erstellen Standard-Live-Aufnahmeprofile Dokument, um festzustellen, ob ein vorhandenes Profil verfügbar ist, das Ihren Anforderungen entspricht.

Anmerkungen:
  • Jedes Profil kann maximal 25 Wiedergaben erstellen.
  • Sie können Erstellen Sie Wiedergaben für vertikale Videos - schließen Sie einfach ein height und width Werte in den Wiedergabespezifikationen, die mit dem Seitenverhältnis des Videos übereinstimmen (1080 Pixel ist der maximal zulässige Wert für das height).
  • Wenn Live-Profile in Studio unter . ausgeblendet sind Administrator > Aufnahmeprofile , diese sind nicht im auswählbar Live-Event erstellen UI, können aber weiterhin mit der Live-API verwendet werden.
  • Live-Stufen:
    • Damit ein Live-Aufnahmeprofil als Tier 1 eingestuft wird, darf die Summe der Bitraten aller seiner Wiedergaben nicht größer als 5.450 sein.
    • Wenn die Summe der Bitraten aller seiner Wiedergaben größer als 5450, aber kleiner als 10000 ist, wird das Live-Aufnahmeprofil als Tier 2 betrachtet.
    • Wenn die Summe der Bitraten aller seiner Wiedergaben größer als 10000 ist, wird das Profil als Tier 3 betrachtet.
Jedes Profil kann maximal 25 Formatvarianten erstellen.

Neues Profil erstellen

Auf der Seite Verwaltung von Aufnahmeprofilen können benutzerdefinierte Live-Aufnahmeprofile erstellt werden. Beachten Sie, dass das Profil in der Benutzeroberfläche des Live-Moduls angezeigt werden soll "model_version": 1 als Teil der Profileigenschaften.

So erstellen Sie ein neues benutzerdefiniertes Live-Aufnahmeprofil:

  1. Drücke den ADMINISTRATOR Link im Studio-Header.
  2. Drücke den Profile aufnehmen Verknüpfung.
  3. Klicken Neues Profil> Live.
  4. Geben Sie ein Name für das Profil und klicken Sie dann Erstellen.
  5. Verwenden Sie den Profileditor, um das Profil nach Bedarf zu bearbeiten.
  6. Klicken speichern.

Profil duplizieren

Sie können ein neues Profil erstellen, indem Sie ein vorhandenes Profil duplizieren. Anschließend können Sie das neue Profil mit Ihren Änderungen bearbeiten. Beachten Sie, dass das Profil in der Benutzeroberfläche des Live-Moduls angezeigt werden soll "model_version": 1 als Teil des Profils.

So erstellen Sie ein benutzerdefiniertes Live-Aufnahmeprofil durch Duplizieren eines vorhandenen Profils:

  1. Drücke den ADMINISTRATOR Link im Studio-Header.
  2. Drücke den Profile aufnehmen Verknüpfung.
  3. Suchen Sie ein Profil mit a Art von Standard Live zu duplizieren. Klicken Sie auf den Profilnamen, um den Profileditor zu öffnen.
  4. Klicken Duplikat. Der Profileditor wird mit dem neuen Profil geöffnet.
  5. Scrollen Sie zum Anfang des Profils. Bearbeiten Sie die name Element, um dem Profil einen aussagekräftigen Namen zu geben. Dieser Name wird in Video Cloud Studio angezeigt.
  6. Bearbeiten Sie die description Element, um dem Profil eine aussagekräftige Beschreibung zu geben.
  7. Bearbeiten Sie die Wiedergabeinformationen nach Bedarf.
  8. Klicken speichern.

Das neu erstellte Profil wird in der Liste der Profile angezeigt. Benutzerdefinierte Profile können gelöscht werden, indem Sie auf das Löschen () Taste.
